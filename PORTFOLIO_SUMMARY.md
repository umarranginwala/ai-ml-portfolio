## 🚀 Quick Start - Deploy to GitHub Pages

### 1. Create GitHub Repository
Go to https://github.com/new and create a repository named `umarranginwala.github.io`

### 2. Push Your Code
```bash
cd /Users/umarranginwala/Documents/GU
git init
git add .
git commit -m "Initial portfolio commit - M.Sc AI/ML Thesis"
git remote add origin https://github.com/umarranginwala/umarranginwala.github.io.git
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to repository → Settings → Pages
2. Source: Deploy from a branch → Select "main" → Save
3. Your site will be live at: **https://umarranginwala.github.io**
