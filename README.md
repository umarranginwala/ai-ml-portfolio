## Projects

### 1. Stock Price Prediction Using Deep Learning
**Master's Thesis Project | NSE (National Stock Exchange of India)**

[![Python](https://img.shields.io/badge/Python-3.8+-3776AB?style=flat&logo=python&logoColor=white)](https://python.org)
[![TensorFlow](https://img.shields.io/badge/TensorFlow-2.x-FF6F00?style=flat&logo=tensorflow&logoColor=white)](https://tensorflow.org)
[![Keras](https://img.shields.io/badge/Keras-DL-D00000?style=flat&logo=keras&logoColor=white)](https://keras.io)
[![Pandas](https://img.shields.io/badge/Pandas-Data-150458?style=flat&logo=pandas&logoColor=white)](https://pandas.pydata.org)

**Abstract:**
Developed and compared multiple deep learning architectures (LSTM and GRU) for predicting stock prices of NSE-listed companies. The research achieved significant accuracy in forecasting Hindustan Unilever Ltd. (HINDUNILVR.NS) stock prices using historical data from 2000-2024.

**Key Achievements:**
- ✅ Achieved high prediction accuracy on 24+ years of historical stock data
- ✅ Implemented and compared LSTM vs GRU architectures
- ✅ Used Yahoo Finance API for real-time data collection
- ✅ Comprehensive EDA with technical indicators
- ✅ Published research paper on stock market prediction

**Technologies:** Python, TensorFlow, Keras, LSTM, GRU, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Yahoo Finance API

**[📁 View Project Details](./projects/stock-prediction/README.md)** | **[📊 Live Demo](./docs/stock-prediction.html)** | **[📄 Research Paper](./Research%20Paper/NSE_stock_price_prediction_umar.pdf)**

### 2. Oral Cancer Detection Using Deep Learning
**Healthcare AI Project | Computer Vision**

[![Python](https://img.shields.io/badge/Python-3.8+-3776AB?style=flat&logo=python&logoColor=white)](https://python.org)
[![TensorFlow](https://img.shields.io/badge/TensorFlow-2.x-FF6F00?style=flat&logo=tensorflow&logoColor=white)](https://tensorflow.org)
[![Keras](https://img.shields.io/badge/Keras-DL-D00000?style=flat&logo=keras&logoColor=white)](https://keras.io)
[![CNN](https://img.shields.io/badge/CNN-Computer%20Vision-00BCD4?style=flat)](https://en.wikipedia.org/wiki/Convolutional_neural_network)

**Abstract:**
Built an automated oral cancer detection system using Convolutional Neural Networks (CNN) and Transfer Learning with ResNet50. The system classifies oral cavity images to assist in early cancer detection, potentially saving lives through early diagnosis.

**Key Achievements:**
- ✅ Implemented CNN from scratch for medical image classification
- ✅ Applied Transfer Learning with ResNet50 architecture
- ✅ Achieved high accuracy with F1 score optimization
- ✅ Created Flask web application for real-time predictions
- ✅ Data augmentation for improved model generalization

**Technologies:** Python, TensorFlow, Keras, CNN, ResNet50, Transfer Learning, Flask, OpenCV, ImageDataGenerator

**[📁 View Project Details](./projects/oral-cancer-detection/README.md)** | **[🏥 Live Demo](./docs/oral-cancer.html)**
